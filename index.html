The Box Model
Section Name Format Use Description
Content This is an example of Content in the Box Model. "Content" is inside of an element being effected by the Box Model.
Padding padding:value; "Padding" is the space between the content and border.
Border border:value; "Border" is space between the padding and margin.
Margin margin:value; "Margin" is the space between the border and other elements.
Offset property Description
XXX-Left Used with Padding, Border, or Margin to offset the selected from the left.
XXX-Right Used with Padding, Border, or Margin to offset the selected from the right.
XXX-Top Used with Padding, Border, or Margin to offset the selected from the top.
XXX-Bottom Used with Padding, Border, or Margin to offset the selected from the bottom.
